This installer will work only if you have installed CentOS version 6.9 64 bit Minimal OS on your linux host.

**********************************************************************************************************************
IMPORTANT : CentOS 6.9 64 bit installation MUST be Minimal. This installer will fail if OS installation is FULL or Custom.
**********************************************************************************************************************

If you don't have CentOS 6.9 64 Bit Minimal ISO, please download it via following URL

http://centos.mirror.net.in/centos/6.9/isos/x86_64/CentOS-6.9-x86_64-minimal.iso

Before running the installer, please fill the IP Address details in installer configuration file appinstallerconf.txt

Please read the documentation properly before proceeding with the installation.
